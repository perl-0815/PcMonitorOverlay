PC Monitor Overlay
==================

このフォルダの内容をまとめて配布してください。

インストール:
  PcMonitorOverlayInstaller.exe を実行してください。

アンインストール:
  スタートメニューの "Uninstall PC Monitor Overlay" を実行するか、
  PcMonitorOverlayUninstaller.exe を実行してください。

直接起動:
  インストールせずに試す場合は PcMonitorOverlay.exe を実行してください。

補足:
  - 配布版は self-contained です。実行だけなら .NET Runtime / SDK は不要です。
  - インストール先は %LOCALAPPDATA%\Programs\PcMonitorOverlay です。
  - 設定は %LOCALAPPDATA%\PcMonitorOverlay\settings.json に保存されます。
